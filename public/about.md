# About Tokamak Network

**Every app deserves its own L2.**

Tokamak Network is an Ethereum-native Layer 2 platform that enables on-demand rollup deployment. Built on the OP Stack, it allows any application to launch its own optimized L2 chain in minutes.

## What We Build

### Thanos L2
On-demand rollup infrastructure with custom chain configuration, one-click deployment, and an open modular architecture. Deploy a production-ready L2 with 1.2s block time through the [Rollup Hub](https://rolluphub.tokamak.network/).

### Seigniorage Staking
TON holders stake tokens to earn seigniorage rewards while securing the network. Over 42M TON is currently staked across the ecosystem.

### Cross-Chain Bridge
Trustless asset transfers between Ethereum L1 and Tokamak L2 chains in under 3 minutes using the cross-trade protocol.

### DAO Governance
On-chain proposal and voting system where TON stakers participate in protocol decisions through the DAO Committee.
